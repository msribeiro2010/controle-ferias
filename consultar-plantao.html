Consultar e Atualizar Plantão

Instruções

Esta página permite consultar e atualizar a data de um plantão no banco de dados Firebase.

O script irá:

  1. Listar todos os plantões do usuário autenticado
  2. Solicitar o ID do plantão que deseja atualizar
  3. Atualizar a data do plantão para 03/03/2025
  4. Listar novamente os plantões para confirmar a atualização

Abra o console do navegador (F12) para ver os resultados.
